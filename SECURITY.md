# 🛡️ Security Policy

## Supported Services
Because AméthMC operates as a live suite of web utilities, only the current production version is actively supported with security updates. 

| Service / Tool | Supported |
| -------------- | :-------: |
| MRPACK Converter | ✅ |
| AméthMC Web Interface | ✅ |
| Forge Master Admin Portal | ✅ |

## Reporting a Vulnerability
If you discover a critical security flaw, vulnerability, or exploit within any AméthMC tool, **please do not open a public GitHub issue.** We need to keep the Forge safe for all explorers while the matrix is being patched.

Instead, please report the vulnerability directly via email:
📧 **support@frisly.me**

### What to include in your report:
* A clear description of the vulnerability and its potential impact.
* Exact, step-by-step instructions to reproduce the exploit.
* Any relevant payloads, screenshots, or console logs.

### What happens next?
1. You will receive an acknowledgment of your report within 48 hours.
2. The vulnerability will be investigated and patched in the staging environment.
3. Once the patch is pushed to the live AméthMC website, we will notify you!

*Thank you for helping keep the AméthMC ecosystem secure!*
