# 🤝 Contributing to AméthMC

First off, thank you for considering contributing to AméthMC! It's people like you that make this community great. 

Since the core source code for AméthMC's web utilities is kept private, this repository is dedicated to **community support, bug tracking, and documentation**.

Here is how you can help keep the Forge running smoothly:

## 📖 Improving the Documentation
Found a typo in the ancient texts? Know a better way to explain the MRPACK Converter? We welcome Pull Requests (PRs) to improve our `docs/` folder!

### How to submit a change:
1. **Fork** this repository to your own GitHub account.
2. **Clone** it to your local machine or edit it directly in the GitHub browser.
3. **Create a branch** for your edits (e.g., `git checkout -b fix/converter-typo`).
4. **Make your changes** to the Markdown files in the `docs/` directory.
5. **Commit your changes** with a clear, descriptive message (e.g., `Fixed spelling error in getting-started.md`).
6. **Open a Pull Request** against the `main` branch of this repository.

The Forge Master (DevFrisly) will review your changes. Once approved, your contributions will be merged into the official archives!

## 🐛 Submitting Excellent Bug Reports
You don't need to write code to contribute. Submitting highly detailed bug reports is one of the best ways to help out. 
When using the Bug Report template, please provide:
* The exact steps needed to reproduce the error.
* Screenshots of the console or the error message.
* Your browser and OS details.

## 💡 Suggesting Features
Got an idea for the next big Minecraft utility? Open a Feature Request! 
Please be as detailed as possible. Mockups, sketches, and clear explanations of the problem your tool solves make it much easier to turn your idea into reality.

---

### Code of Conduct
By participating in this project, you agree to keep the community welcoming and respectful. Harassment, spam, or toxic behavior in the issues or pull requests will result in a ban from the repository.

*Thank you for helping us build better tools for the Minecraft community!*
